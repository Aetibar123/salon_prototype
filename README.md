# 💎 Luxury Salon Website
link: https://salon-prototype-kohl.vercel.app/
A premium, modern, and fully responsive **Luxury Salon Website** built with **Next.js**, **TypeScript**, and **Tailwind CSS**. This project showcases a high-end UI/UX experience for salons, spas, and beauty brands with elegant layouts, smooth animations, immersive media, and reusable components.

> **Note:** This is a frontend prototype built for demonstration purposes. All forms simulate successful submissions without a backend.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/luxury-salon-website.git
```

Navigate to the project:

```bash
cd luxury-salon-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```
